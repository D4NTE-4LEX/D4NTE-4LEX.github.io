¿Qué son las media queries?

Es una regla, introducida en CSS2, para aplicar estilos CSS dependiendo del tipo general de dispositivo u otra características como la resolución de pantallao el ancho del viewport del navegador
Las media queries se pueden usar para comprobar muchas cosas, como:
  • Ancho y alto del viewport
  • Ancho y alto del dispositivo
  • Orientación (horizontal o vertical)

Su sintaxis

Una media query consiste en un tipo de medio y puede contener una o más expresiones, que se resuelven como verdadero o falso
La sintaxis de una media query es la siguiente:

@media not|only mediatype and (expressions) { CSS-Code; }

Tipos de Media

Hay cuatro tipos de media:
Valor Descripción
all Usado para todos los tipos de dispositivos
print Usados para impresoras
screen Usado para todos los dispositivos con pantalla como computadoras, tabletas, telefonos inteligentes, etc.
speech Usado para leedores de pantalla
